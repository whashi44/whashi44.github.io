Lindemann Index Calculator by Python(Github)

GUI to calculate lindemann index

This python script calculates a lindemann index, which is defined as follows: \[\delta =\frac{2}{N(N-1)} \sum_{i \lt j}\sqrt{\frac{\langle r_{ij}^2 \rangle_t-\langle r_{ij}\rangle_t^2}{\langle r_{ij}\rangle_t}} \]


I had to calculate this value from molecular dynamics simulation dump file and after quick search, I could not find a software or code that does this with my specific condition, so I created one.

Then, I got tired of changing the parameters within the code for specific scenarios, so I created a GUI for flexibility, which can be seen in the following GIF.

Finite Element Analysis by Matlab(Github)

Approximation of 2nd order partial differential equation using matlab

This MATLAB code approximates the solution for the following partial differential equation:
\[\frac{\partial p}{\partial t} - \nabla\cdot(\bf{K}\nabla p) = q\] by converting the strong form to weak form using divergent theorem, and approximating the solution using discretization, shape function, Gallerkin's approximation, and Gaussian quadrature. The resulting animation is below, which shows the approximate solution being updated every time step to obtain theoretical solution.
